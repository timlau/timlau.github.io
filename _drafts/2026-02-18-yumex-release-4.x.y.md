## Description
Yum Extender 4.x.y has just been released!

Yum Extender is a powerful tool for managing software packages on Fedora Linux. It provides a user-friendly interface for searching, installing, updating, and removing packages and Flatpaks.

## Installation
The feature is available in the latest development version of Yum Extender available in the [yumex-ng ](https://copr.fedorainfracloud.org/coprs/timlau/yumex-ng/) COPR Repository

To install it, run the following commands:

```terminal
sudo dnf copr enable timlau/yumex-ng
sudo dnf install yumex
```
